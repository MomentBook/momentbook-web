# momentbook-web

Official web for **MomentBook** — landing, product pages, download links, and SEO-friendly content.

> This repo is intended to be the public-facing website (not the mobile app, not the API).

---

## Tech Stack

-   Next.js (App Router recommended)
-   TypeScript (optional, depending on project)
-   i18n routing (e.g. `/en`, `/ko`, `/ja`, `/zh`)
-   Deployment: Vercel / Cloudflare Pages / self-hosted (Nginx) — choose one

---

## Getting Started

### Requirements

-   Node.js 20+
-   Package manager: `yarn` (or npm/pnpm)

### Install

```bash
yarn install
```
